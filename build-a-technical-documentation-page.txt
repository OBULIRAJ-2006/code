      <header>Introduction to JavaScript</header>
      <p>JavaScript is a versatile programming language primarily used for web development. It allows developers to add interactivity to web pages.</p>
      <p>It is a client-side scripting language but can also be used on the server side with Node.js.</p>
      <p>JavaScript is essential for building dynamic, responsive web applications.</p>
      <code>console.log("Hello, World!");</code>
      <ul>
        <li>First appeared: 1995</li>
        <li>Developed by: Brendan Eich</li>
        <li>Standardized by: ECMA International</li>
      </ul>
    </section>

    <!-- Section 2 -->
    <section class="main-section" id="Variables_and_Data_Types">
      <header>Variables and Data Types</header>
      <p>JavaScript supports multiple data types, including numbers, strings, and objects.</p>
      <p>Variables in JavaScript can be declared using <code>var</code>, <code>let</code>, or <code>const</code>.</p>
      <p>Data types in JavaScript are broadly categorized as primitive and non-primitive.</p>
      <code>let age = 25;</code>
      <ul>
        <li>Primitive types: string, number, boolean, null, undefined, symbol</li>
        <li>Non-primitive types: object, array, function</li>
      </ul>
    </section>

    <!-- Section 3 -->
    <section class="main-section" id="Functions_in_JavaScript">
      <header>Functions in JavaScript</header>
      <p>Functions are reusable blocks of code designed to perform a specific task.</p>
      <p>Functions can be declared or expressed as anonymous or arrow functions.</p>
      <p>JavaScript supports first-class functions, allowing them to be treated as variables.</p>
      <code>function greet(name) { return Hello, ${name}; }</code>
      <ul>
        <li>Function declaration</li>
        <li>Anonymous function</li>
        <li>Arrow function</li>
      </ul>
    </section>

    <!-- Section 4 -->
    <section class="main-section" id="Loops_and_Iteration">
      <header>Loops and Iteration</header>
      <p>Loops are used to execute a block of code repeatedly based on a condition.</p>
      <p>JavaScript provides several looping constructs: <code>for</code>, <code>while</code>, <code>do-while</code>, and more.</p>
      <p>Iterators like <code>for...of</code> and <code>for...in</code> allow traversal over objects and arrays.</p>
      <code>for (let i = 0; i < 5; i++) { console.log(i); }</code>
      <ul>
        <li>For loop</li>
        <li>While loop</li>
        <li>Do-while loop</li>
      </ul>
    </section>

    <!-- Section 5 -->
    <section class="main-section" id="ES6_Features">
      <header>ES6 Features</header>
      <p>ECMAScript 6 (ES6) introduced many new features to JavaScript, making it more powerful and efficient.</p>
      <p>Key features include arrow functions, template literals, classes, and modules.</p>
      <p>ES6 also added <code>let</code> and <code>const</code> for variable declarations.</p>
      <code>const add = (a, b) => a + b;</code>
      <ul>
        <li>Arrow functions</li>
        <li>Template literals</li>
        <li>Classes and inheritance</li>
